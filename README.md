## Installation

### Clone the repo

    $ git clone https://github.com/putaindecode/ed-209

### NPM dependencies

    $ make install

### Install and run 

    $ make

### Copy config

    $ cp config.json.dist config.json

### Modify the config

Please, change my configuration (`config.json`).

### Start me!

    $ make bot
